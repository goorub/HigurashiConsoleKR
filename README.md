## 콘솔 에피소드 읽는 순서

- 소메우츠시 -> 카게보우시
- (츠미호로보시 배드엔딩 이후) 요이고시
- (미나고로시 이후) 토키호구시 -> 조각의 세계 -> 미오츠쿠시 우라
- 이후 코토호구시 -> 에필로그

- (사이코로시 이후) 하지사라시

## 미오츠쿠시 우라 / 미오츠쿠시 오모테 순서에 대해

- 어떻게 읽어도 상관 없습니다.
- DS버전처럼 감상하려면, 오모테10일차까지 -> 우라 -> 오모테 나머지 부분을 읽습니다. ([출처](https://wiki.07th-mod.com/Higurashi/MiotsukushiFAQ/#what-order-do-i-read-these-arcs-in))
